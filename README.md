## ⚡ First-Time Setup

### Step 1 — Install uv (once, globally)

**Windows (PowerShell):**
```powershell
irm https://astral.sh/uv/install.ps1 | iex
```
**Mac / Linux:**
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

> uv replaces pip + virtualenv + requirements.txt. It reads `pyproject.toml` and creates `uv.lock` automatically.

---

### Step 2 — Install project dependencies

**Windows:**
```powershell
.\do.ps1 install
```
**Mac / Linux:**
```bash
./do install
```

This runs `uv sync` which:
1. Creates a `.venv` in the project folder
2. Installs all packages from `pyproject.toml`
3. Auto-generates `uv.lock` (exact version lockfile — commit this to Git)

---

### Step 3 — Add your API key

```powershell
copy .env.example .env
# Open .env in Notepad and set: ANTHROPIC_API_KEY=sk-ant-...
```

---

### Step 4 — Run the app

**Windows:**
```powershell
.\do.ps1 app
```
**Mac / Linux:**
```bash
./do app
```

Then open `http://localhost:8501` and upload your Octoparse CSVs via the sidebar.

## Why uv.lock and no requirements.txt?

| Old way | uv way |
|---------|--------|
| `pip install -r requirements.txt` | `uv sync` |
| `requirements.txt` (manual, drifts) | `pyproject.toml` (source of truth) |
| No exact version lock | `uv.lock` (auto-generated, exact versions) |
| Different versions on each machine | Identical installs everywhere |

**Rule:** edit `pyproject.toml` to add/remove packages. Never edit `uv.lock` — uv regenerates it. Commit both files to Git.

**Why not `python app.py` directly?** That uses your system Python which has none of the packages. Always use `.\do.ps1 app` which calls `uv run` internally.

## Octoparse CSV Format

**restaurants.csv:** `Page_URL, name, address, rating, review_count, website, phone, price`

**reviews.csv:** `Page_URL, reviewer_name, review_date, review_rating, review_text, owner_response, owner_response_content`

Column names are auto-detected — minor differences are handled.

## Scoring

| Dimension | Weight | Formula |
|-----------|--------|---------|
| Reputation | 30% | `(rating/5)×70 + min(reviews/500,1)×30` |
| Responsiveness | 25% | `replied / total × 100` |
| Digital Presence | 20% | website(50) + phone(25) + base(15) + price |
| Intelligence | 15% | `((avg_rating−1)/4) × 100` |
| Visibility | 10% | recency-weighted: `(90d×0.7 + 180d×0.3) / total` |

**Silent Winner** = rating ≥ 4.5 AND response rate < 30%
